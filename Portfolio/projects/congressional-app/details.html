Congressional App - Golf Guru Demo

Problem

This interactive app brings a golf training experience to the web, helping users track profile data, club distances, recommended club selection, and scorecard performance.

Solution

The demo recreates the core Golf Guru experience using a lightweight browser interface. Users can save profile details, enter club yardages, request a club recommendation for a hole distance, and track scores.

Tech stack

  • React-inspired browser demo
  • JavaScript
  • HTML / CSS

Key features

  • Profile setup
  • Club distance planner
  • Distance-to-hole recommendation
  • Scorecard tracker with par totals
  • Source code included from the original React app

Reflection

This project displays UX-focused planning and stateful app behavior. It proves I can transform a React design into a portfolio-ready interactive demo while preserving code accessibility and source transparency.
