Project Details

EMS Urgency Predictor

Problem

This project predicts urgency levels for emergency patients from vital signs and symptoms. It supports faster triage decisions by estimating whether a case is Low, Medium, or High urgency.

Solution

A Decision Tree model is trained using sample EMS data. The browser demo uses a heuristic that mirrors the model's behavior, allowing visitors to test patient inputs instantly and understand the prediction flow.

Tech stack

  • Python
  • Scikit-learn
  • Pandas
  • HTML / CSS / JavaScript

Key features

  • EMS symptom and vital input form
  • Urgency prediction result display
  • Downloadable Python training script

Reflection

This project demonstrates applied machine learning for healthcare. It shows how to build a model from data, evaluate it, and then present a user-friendly demo that explains the prediction experience.
