Project Details

Classroom Finder

Problem

This app solves the problem of quickly finding teachers and rooms in a school directory. It replaces manual searching with a searchable interface that supports name, room, and building queries.

Solution

The application loads local teacher data from a text file and lets users search by multiple fields. It displays matching results in cards and provides count feedback for each query.

Tech stack

  • JavaScript
  • HTML / CSS
  • Client-side data parsing

Key features

  • Search by teacher name, room number, or building
  • Result cards with contact details
  • Clear search and count of matching results

Reflection

This project demonstrates how to turn raw text data into a searchable interface. It highlights front-end data processing, UI feedback, and building a polished directory experience.
