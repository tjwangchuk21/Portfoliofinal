Project Details

Deadly Desert Map

Problem

This project analyzes a desert grid and identifies dangerous locations where the values around a cell indicate a deadly trap. It helps visualize risk across a map and count the total dangerous points.

Solution

The web demo loads a static grid, computes adjacent-value sums, and marks cells as deadly when the sum exceeds a threshold. The browser output shows both the original numeric map and the deadly map together.

Tech stack

  • Java (original logic)
  • JavaScript (browser demo)
  • HTML / CSS (UI)

Key features

  • Interactive threshold control
  • Side-by-side original and deadly map views
  • Deadly spot counter
  • Source code available in `Main.java`

Reflection

This project demonstrates how to translate algorithmic map analysis into a user-friendly visualization. It reinforces array processing, neighbor-based decision logic, and presenting results clearly for end users.
